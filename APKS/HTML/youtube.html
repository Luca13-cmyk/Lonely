Isso é um pequeno aplicativo web para ver videos sem travar, do youtube. Ele salva os dados automaticamente. Se quiser limpar, no menu tem a opção limpar dados. Ele não foi trabalhado para ser responsivo, por isso, só deve ser usado em telas com boa largura
